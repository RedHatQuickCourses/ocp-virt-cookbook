== Usage

=== Download and Apply

. Click the YAML link to download the manifest
. Customize values for your environment
. Apply with `oc apply -f <filename>.yaml`

=== Validate Before Applying

[source,bash,role=execute]
----
oc apply --dry-run=client -f <filename>.yaml
----
