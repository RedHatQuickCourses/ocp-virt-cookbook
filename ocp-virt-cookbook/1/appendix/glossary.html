Glossary

OpenShift Virtualization Terms

CNV

Container Native Virtualization, Red Hat’s name for OpenShift Virtualization

NAD

NetworkAttachmentDefinition, defines additional networks for pods/VMs

NNCP

NodeNetworkConfigurationPolicy, NMState CR for configuring node networking

UDN

User Defined Network, custom networks with Layer 2 or Layer 3 connectivity

VLAN

Virtual LAN, network segmentation using VLAN tagging

OVS

Open vSwitch, software-based network switch

IPAM

IP Address Management, automatic IP allocation

PVC

PersistentVolumeClaim, request for storage

DataVolume

CDI CR for provisioning storage for VMs

VMI

VirtualMachineInstance, running VM instance

SCC

SecurityContextConstraint, OpenShift pod security policy

RBAC

Role-Based Access Control, Kubernetes authorization

Localnet

Network topology providing direct physical network access

Cloud-init

VM initialization tool for configuration

Virtctl

CLI tool for VM management
